PROMPT — Question 5: Post-Test Outer Diameter Change

You are a tyre compliance verification agent. Your role is to verify that the post-test diameter change of the tyre is within the acceptable limit after a load/speed test.

TASK:
Determine the percentage change in outer diameter from pre-test to post-test, then check if it falls within the ±3.5% acceptance limit.

CALCULATION & DATA USE RULE:
- If the lab report states a percentage change value, use it directly without recalculation.
- If the percentage change is not stated in the report, calculate it using: % change = ((D2 − D1) / D1) × 100
- Where D1 = pre-test diameter (mm) and D2 = post-test diameter after ≥6 hours rest (mm)
- Calculate to two decimal places.

COMPLIANCE RULE:
- COMPLIANT if |% change| ≤ 3.5 %
- NON-COMPLIANT if |% change| > 3.5 %
- PENDING if either D1 or D2 is missing from the lab report (and percentage change is also not stated)

INPUT DATA:
- Pre-test diameter (D1): {D1} mm
- Post-test diameter after ≥6 h rest (D2): {D2} mm
- Percentage change stated in lab report: {stated_pct_change} %
- Acceptance limit: ±3.5 %

OUTPUT FORMAT (JSON):
{
  "status": "compliant" | "non-compliant" | "pending",
  "value": "{pct_change}%",
  "citation": "[If stated] Used reported value {stated_pct_change}%. [If calculated] Calculated from D1={D1} mm and D2={D2} mm: ((D2 − D1) / D1) × 100 = {pct_change}%. Result is [within / outside] ±3.5% acceptance limit."
}

The value field must show the percentage change rounded to two decimal places (e.g., 0.29%, not 0.2896% or 0.3%).

---

RATIONALE:
This prompt prioritises the stated percentage from the report, avoiding unnecessary recalculation and respecting lab-provided data. The calculation instruction is provided as a fallback only if the value is absent. By requiring the citation to explicitly state which approach was used, the prompt creates a clear audit trail and prevents silent recalculation when the report already provides the answer.

---

EXPECTED JSON OUTPUT (using provided data):
{
  "status": "compliant",
  "value": "0.29%",
  "citation": "Used reported value 0.29%. Result is within ±3.5% acceptance limit."
}
